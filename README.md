# Inventory Control API

This is a study project created to learn C#, .NET, Web API, and Clean Architecture.

The goal is to build an inventory control API step by step, separating responsibilities into clear layers:

- Domain
- Application
- Infrastructure
- API

## Technologies

- C#
- .NET
- ASP.NET Core Web API
- Clean Architecture
